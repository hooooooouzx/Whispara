# Whispara

**The Cognitive Perception Engine: Capture the Whisper, Follow the Shadow.**

A revolutionary landing page for Whispara, the world's first true Intent Router.

## Features

- ✨ Modern, unique UI design with smooth animations
- 🎯 Interactive four-dimensional interaction showcase
- 🌊 Responsive design that works on all devices
- 🧠 Dark mode support
- ⚡ Built with Next.js 16 and Tailwind CSS
- 🎨 Custom animations and gradient effects

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Custom CSS animations

## Philosophy

Whispara transforms the traditional note-taking paradigm:

- **From Archive to Portal**: Not a final destination, but the starting line of thought
- **Zero-Directory Architecture**: Notes find you, not the other way around
- **Neural Network Model**: Auto-aggregation instead of manual filing
- **Passive Awakening**: Context-aware semantic retrieval

## Four Dimensions

1. **The Whisper** - Instant Capture Mode (Home)
2. **The Flow** - Gravity Directory (Swipe Right)
3. **The Shadow** - Echo Mode (Swipe Left)
4. **The Dispatcher** - Intent Mapping (Swipe Down)

## Build for Production

```bash
npm run build
npm start
```
